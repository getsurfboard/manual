# 安装

你可以通过以下渠道获取 Surfboard 应用：

- Telegram 频道: [https://t.me/surfboardnews](https://t.me/surfboardnews)
- APKPure: [https://apkpure.com/surfboard/com.getsurfboard](https://apkpure.com/surfboard/com.getsurfboard)
- Github 发行版: [https://github.com/getsurfboard/surfboard/releases/latest](https://github.com/getsurfboard/surfboard/releases/latest)

:::tip 提示
- APKPure 和 Telegram 频道分发的应用通常是相同的。

- APKPure 分发的版本通常相对稳定，但更新频率较低。

- Telegram 频道更新更为频繁，但可能存在未知问题。

一般来说，经过 Telegram 频道验证的稳定版本会定期发布到 APKPure。

你可以根据实际需求选择安装和更新渠道。
:::
