# 最终规则

`FINAL` 规则作为兜底策略，用于处理配置文件中未匹配到任何前面规则的所有流量。

通常，一个配置文件应该只包含一条最终规则，并且必须将其放置在规则部分的最后一条。

## 示例

```ini
# 将所有其它未匹配的流量直接转发
FINAL, DIRECT
```

## 格式

```ini
FINAL, {target}
```

## 参数

| 名称         | 值 | 是否必填 | 备注                                                   |
|--------------|-------|-----------|--------------------------------------------------------|
| target       | -     | 是      | 指定的代理或策略组必须存在于配置文件中。 |
