# 深层链接

## 从 URL 导入配置文件

```    
surfboard:///install-config?url=[encoded_profile_url]
```

## 切换 VPN 状态 (Toggle)

```  
surfboard:///toggle
```

## 开启 VPN

```  
surfboard:///start
```

## 关闭 VPN

```
surfboard:///stop
```

:::tip 提示
也支持部分 [Surge 的深层链接格式](https://manual.nssurge.com/others/url-scheme.html)，只需将 `surfboard:///` 更改为 `surge:///` 或 `surge3:///`，一切将照常运行。
:::
